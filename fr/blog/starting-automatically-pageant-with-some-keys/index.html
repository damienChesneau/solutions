Démarrer Pageant automatiquement avec des clés.

  • Par Damien Chesneau
  • 2 Décembre 2019
  • SSH

On utilise tous les jours des logiciels qui nécessitent une configuration certes facile à mettre en œuvre, mais répétitive. Par exemple pour la connexion aux machines virtuelles dans le cloud, j'utilise Putty sa configuration est assez simple. Pour mon utilisation, je n'ai qu'à spécifier le nom de la machine ou son adresse IP (par exemple : root@192.168.1.1) ainsi que la clé privée pour l'authentification.

Sauf que dans le contexte du cloud nous avons créons/détruisons beaucoup machines virtuelles et que les clés privées associés peuvent être différentes il est souvent fastidieux de trouver la clé qui correspond à la VM qui vient d'être créé.

C'est pourquoi j'ai créé le script ci-dessous permettant de lancer Pageant avec un ensemble de clés contenu dans un dossier. Et en le plaçant dans le dossier du “shell:startup” (Action Windows+R) cela permet qu'il se lance directement au démarrage de Windows.
