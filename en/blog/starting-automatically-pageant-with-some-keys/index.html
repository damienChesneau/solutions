Starting automatically Pageant with some keys.

  • By Damien Chesneau
  • December 2 2019
  • SSH

Some software is used every day which requires a configuration that is certainly easy to implement, but repetitive. For example for connecting to virtual machines in the cloud, I use Putty because its configuration is quite simple. For my use, I just have to specify the name of the machine or its IP address (for example: root@192.168.1.1) as well as the private key for authentication.

Except that in the context of the cloud I have to create/destroy many virtual machines per day and that the associated private keys can be different It is often tedious to find the key that corresponds to the VM that has just been created.

That's why I created the script below to launch Pageant with a set of keys contained in a folder. And by placing it in the folder "shell:startup" (Action Windows+R) it allows it to launch directly at Windows startup.
